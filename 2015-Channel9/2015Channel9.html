F12 Chooser

F12 Chooser

A tool to debug IE wherever it may be
  • Stand alone exe in the Program Files/Internet Explorer/F12Chooser.exe
  • Used to launch the F12 tools against IE, works against:
    • Native apps hosting a WebOCs
    • Managed apps using the .net browser control (WinForms, WPF, etc.)
    • HTA
    • IE
  • All the tools are the same as when F12 is launched against IE
